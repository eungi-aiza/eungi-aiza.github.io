About this template

This minimal static site is designed for GitHub Pages and aims to mimic a Notion-style two-column layout where navigation lives in a fixed sidebar on the left and content appears on the right. You can use it as a starting point for simple blogs, notebooks, or documentation.

Usage

  1. Clone or fork the repository to your own GitHub account.
  2. Edit the HTML files (e.g. `index.html`, `machine-learning.html`) to add your own content. Every page should include the sidebar markup so navigation is consistent.
  3. Change `css/style.css` to customize colours, fonts, or layout. The sidebar width is currently 220Â px; feel free to adjust.
  4. Add new sections by creating new HTML files and linking them in the `
      ` inside the `.sidebar` element on every page.
    • Push changes to the `main` branch and enable GitHub Pages from the repository settings. The site will be published at `https://.github.io//`.

Extending the template

If you'd like more dynamic behavior or templating, consider migrating to a static site generator such as Jekyll, Hugo, or Eleventy. This simple version is easy to understand but doesn't scale well for larger blogs.

For a more Notion-like experience with block editing, you'll need to incorporate JavaScript, a headless CMS, or use Notion's API/embeds; that's beyond the scope of this starter template.
